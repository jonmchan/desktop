# LessPass desktop

LessPass desktop application build with electron

## Dependencies

 * [Node.js/npm](https://nodejs.org/en/download/package-manager/)

```
npm install
```

## Run Locally

    npm start

## Build Linux
 
     npm run build:linux

## Build MacOS
 
     npm run build:osx

## Build Windows
 
     npm run build:windows
